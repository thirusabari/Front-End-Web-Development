HTML Nested List



A list within another list is termed as nested list. If you want a bullet list inside a numbered list then such type of list will called as nested list.



HTML Nested List Example



  1. One
    • Two
  2. Three
    • Four
  3. Five
    • Six
  4. Seven
    • Eight
  5. Nine
    • Ten
  6. Eleven
    • Twelve
