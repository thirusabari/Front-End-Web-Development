HTML JavaScript Introduction


A Script is a small program which is used with HTML to make web pages more attractive, dynamic and interactive,
such as an alert popup window on mouse click. Currently, the most popular scripting language is JavaScript used for websites.


Script Tag


The HTML script tag is used to specify a client-side script.
It may be an internal or external JavaScript which contains scripting statements, hence we can place script tag within
body or head section.

It is mainly used to manipulate images, form validation and change content dynamically. JavaScript uses document.getElementById() method to select an HTML element.
