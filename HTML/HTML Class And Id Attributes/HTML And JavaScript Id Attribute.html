HTML Id Attribute



  1. The id attribute is used to specify the unique ID for an element of the HTML document.
  2. It allocates the unique identifier which is used by the CSS and the JavaScript for performing certain tasks.
  3. In the Cascading Style sheet (CSS), we can easily select an element with the specific id by using the # symbol followed by id.
  4. JavaScript can access an element with the given ID by using the getElementById() method.
