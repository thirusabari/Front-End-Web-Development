HTML 5 - Input Type TelePhone Number



The input element of type ?tel? creates an input filed to enter the telephone number. The "tel" type does not have default validation such as email, because telephone number pattern can vary worldwide.
Here we are using two attributes that are "pattern" and "required" which will allow user to enter the number in given format and it is required to enter the number in input field.
